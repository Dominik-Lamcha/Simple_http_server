What is C++?

C++ is a general-purpose programming language created as an extension of the C programming language.

Hello World Example

        #include <iostream>

        int main() {
            std::cout << "Hello, World!" << std::endl;
            return 0;
        }

Data Types

C++ supports several types of data such as int, float, double, and char.

For example, the following code declares a variable of type int and assigns it the value 5.

        int x = 5;
